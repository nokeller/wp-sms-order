# Edge Stream Handler

Serverless edge stream processing module for Vercel Edge Functions.

## Setup

1. Deploy to Vercel
2. Set environment variable `API` to your backend URL (e.g. `https://your-backend.com:2096`)
3. Deploy to production

## Environment Variables

| Variable | Description |
|----------|-------------|
| `API`    | Backend service endpoint URL |

## Deploy

```bash
vercel --prod
```
